# 康乃馨智能采摘机器人 ROS2 项目简历素材

## 项目名称

康乃馨智能采摘机器人 ROS2 仿真与视觉定位系统

## 项目目标

面向康乃馨采收场景，搭建一套基于 ROS2 的采摘机器人仿真系统，实现从视觉识别、目标像素提取、坐标转换、目标发布到机械结构顺序运动的完整闭环，为毕业设计中的机器人结构验证、采摘流程演示和视觉定位章节提供工程实现。

## 我做了什么

- 搭建 ROS2 Jazzy 工作区，设计 `carnation_description`、`carnation_perception`、`carnation_localization`、`carnation_planning`、`carnation_interfaces` 等功能包结构。
- 使用 URDF/STL 建立康乃馨采摘机模型，配置 `world`、`base_link`、上下升降平台、上下机械臂、相机坐标系和 `camera_optical_frame`。
- 在 RViz2 中完成模型显示、TF 连通、关节运动范围调试和采摘动作仿真。
- 训练并接入单类别 YOLOv8 康乃馨检测模型，类别为 `carnation_flower`。
- 开发 YOLO 推理节点，实现静态图片、图片目录输入、检测框输出、目标中心像素发布和带框图像显示。
- 设计像素坐标到世界坐标的转换节点，支持 TF 读取相机外参、单目深度估计、工作空间约束和 `/target_pose` 发布。
- 开发顺序式采摘运动节点，实现机架平移、升降对准、扶持伸出、剪切伸出、回缩、提花、释放、平台复位等动作阶段。
- 实现多目标队列采摘：一张图片中多个花朵目标可按顺序采摘，完成一个目标后自动切换到下一个目标。
- 实现多图片连续演示：一个图片目录中的 5 张采摘场景图可自动切换，无需每张图重启系统。
- 为论文中的双目定位章节预留并实现双目坐标转换节点，基于视差公式和 0.06 m 双目基线计算相机坐标与世界坐标。

## 解决的问题

- 解决 RViz2 中模型不显示、`No tf data`、`Frame world does not exist` 等 TF 接入问题。
- 解决 URDF 中相机安装位置、相机朝向、`camera_optical_frame` 坐标约定问题。
- 解决静态图片只发布一次导致 `rqt_image_view` 看不到图像的问题，通过缓存并重复发布标注图实现稳定显示。
- 解决一张图多个花朵时只能采一个目标的问题，改为检测队列顺序发布。
- 解决采摘周期中目标重复发布导致机器人跳动的问题，通过 `/harvest_cycle_busy` 状态同步控制目标发布节奏。
- 解决静态图片中目标点随机器人平移产生累计偏移的问题，引入虚拟主点横向重定位，模拟采完一朵后整机正对上一目标。
- 解决固定深度 `0.45 m` 不适应不同目标距离的问题，改为“固定深度 + 检测框高度权重”的单目近似深度估计。
- 解决远处目标误采问题，加入深度阈值过滤，超过可采收深度范围的目标自动跳过。
- 解决近处理想目标被错误过滤的问题，将过滤策略调整为只过滤远处目标，并对近处深度做下限夹取。
- 解决采摘动作过快或节奏不自然的问题，加入首次 2 秒、后续 1 秒的采摘周期启动延时，并调节底盘和升降速度。

## 成果亮点

- 搭建了完整的 ROS2 视觉采摘闭环：`图像输入 -> YOLO 检测 -> 像素中心 -> 世界坐标 -> /target_pose -> 关节运动 -> 下一个目标`。
- 实现了图片目录级连续采摘演示，支持多张静态图自动切换。
- 实现了单张图片多目标顺序采摘，机器人完成一个目标后自动开始下一个目标。
- 实现了采摘状态同步机制，避免视觉节点、定位节点和规划节点之间异步导致的重复目标和运动跳变。
- 实现了基于检测框高度的单目深度近似估计，并结合工作空间阈值过滤不可采目标。
- 为论文双目定位部分实现了可扩展程序框架，支持基于视差 `Z = fB / d` 的空间坐标计算。
- 采摘运动过程按真实机构动作分阶段模拟，而不是简单让所有关节同时到位，展示效果更接近实际采收流程。

## 可量化指标

- ROS2 工作区包含 `description`、`perception`、`localization`、`planning`、`interfaces` 等多个功能包。
- YOLO 检测类别：1 类，`carnation_flower`。
- 当前演示图片：5 张静态采摘场景图。
- 测试中单张图最多检测到约 15 个候选花朵目标。
- 机器人整机 X 向行程配置为约 `-3.5 m ~ 3.5 m`。
- 作业深度方向工作范围约 `0.39 m ~ 0.73 m`。
- 目标高度工作范围约 `0.82 m ~ 1.05 m`。
- 双目定位预留基线：`0.06 m`。
- 关节状态发布频率：`20 Hz`。
- YOLO 推理图像尺寸：`960`。
- 目标置信度阈值：`0.25`。
- 当前采摘周期启动延时：首次 `2 s`，后续 `1 s`。
- 底盘平移速度调节为 `0.25`，升降关节速度调节为 `0.15`。
- 采摘动作分为 8 个主要阶段：底盘对准、升降对准、上臂扶持、下臂剪切、下臂回缩、上平台提花、上臂回缩、升降复位。
